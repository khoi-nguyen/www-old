# Introduction

The **Laplace transform** transforms an IVP in an algebraic equation.

![](/static/images/1686174276.png)

# Laplace transform

::: {.definition title="Laplace transform"}
\begin{align*}
\mathscr L f(s) = \int_0^{+\infty} e^{-st} f(t) \dd t
\end{align*}
:::
